## ✅ Projektin tila
Toteutettu täysin toimivana paikallisena verkkosovelluksena (Node.js + SQLite).
Kaikki vaiheen 1 ja 2 määritellyt toiminnallisuudet on toteutettu.

---

## 🧱 Teknologiat

- **Frontend:** HTML, CSS, JavaScript (vanilla)
- **Backend:** Node.js + Express
- **Tietokanta:** SQLite (`games.db`)
- **Dataformaatti:** JSON (API:n kautta)
- **Projektinhallinta:** GitHub + tuntikirjaus.md

---

## 🎮 Toteutetut toiminnallisuudet

- Pelin lisääminen lomakkeella
- Pelilistan katselu (automaattinen päivitys)
- Pelin poistaminen
- Hakutoiminto
- Yksittäisen pelin tietojen tarkastelu
- Responsiivinen ulkoasu (toimii myös mobiilissa)

## ℹ️ Miksi toteutus on näin?

Koska projekti on vielä perusvaiheessa (MVP), käyttäjäkohtaisia pelilistoja ei ole erikseen tallennettu.
Kaikki pelit tallentuvat yhteiseen listaan, ja käyttäjä voi lisätä omia suosikkejaan pelikohtaisesti.

Kuvauksessa voi mainita esim. “Eliaksen pelit” tai “Toimintapelit”, jolloin pelilista toimii käytännössä käyttäjän kokoamana.

Toteutus on yksinkertainen, mutta tukee hyvin laajentamista tulevaisuudessa:
- Kirjautuminen
- Käyttäjäkohtaiset listat
- Listojen nimeäminen ja jako
